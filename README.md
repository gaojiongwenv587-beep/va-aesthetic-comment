## 核心能力

- **10 种贴文类型自动识别** — 科普清单、求诊照片、情绪共鸣、反论质疑等，按类型对症下藥
- **VA 症状矩阵** — 根据原贴关键字（毛孔、双下巴、35+、痘疤……）只推 1-3 个相关项目
- **完整 VA 价格库** — 韩元原价＋台币换算，覆盖提升/激光/填充/手术全品类
- **语气自然化自检** — emoji 克制、语气词自然、繁简跟随原贴

## 触发词

| 场景 | 示例 |
|------|------|
| 直接写回覆 | VA 评论、VA 留言、帮我回这则贴文 |
| 指定风格 | 整形外科院长口吻、医美 KOL 留言 |
| 引流场景 | 评论引流、诊所留言 |
| 隐式触发 | 贴上他人贴文 + 要求生成回覆/留言/评论 |

## 使用方式

直接贴上目标贴文即可，无需填写 persona：

```
[贴上原贴内文]
帮我写一则回覆
```

## 输出格式

```
【分类】X. 类型名称（信心度：高/中/低）
【推荐项目】1-3 个项目 + 理由
【回覆草稿】
（直接可贴到平台的回覆，含换行、emoji、价格行）
```
