## 年龄段推荐策略

| 年龄段 | 主推方向 |
|--------|----------|
| 20-25岁 | 毛孔（Potenza）、痘疤（Pico）、水光（利朱兰） |
| 25-30岁 | 轻度提升（Shrink）、毛孔、水光、初次填充 |
| 30-35岁 | 提升（Shrink/Density）、填充（法令纹）、Potenza |
| 35-45岁 | Density 或 Ulthera P、热玛吉、填充、线雕 |
| 45岁以上 | Density/Ulthera P 高剂量、拉皮（手术）、综合方案 |
